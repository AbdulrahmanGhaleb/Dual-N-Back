
**Dual N-Back Game**


🚀 **Features**
- **Customizable N-Level** → Choose how many steps back to recall.  
- **Randomized Sequences** → Each game run is different.  
- **Live Score Tracking** → Gain or lose points based on accuracy.  
- **Cross-Platform Support** → Works on Windows, Linux, and macOS.  


🛠 **Technologies Used**
- **C++** → Core game logic.  
- **Object-Oriented Programming (OOP)** → Encapsulation, aggregation, and modular design.  


🎮 **How to Play**
- The game will display a letter at a random position.  
- You must recall if the **current letter or position** matches one from **N rounds ago**.  
- Enter **'y'** for Yes or **'n'** for No.  
- Your score updates based on correct/incorrect responses.
